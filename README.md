# Dotfiles

This repository contains configuration files and settings for various development tools and applications.

## Contents

- **git/** - Git configuration (`.gitconfig`, `.gitattributes`)
- **mise/** - Mise development environment configuration
- **powershell/** - PowerShell profile configuration
- **powertoys/** - Microsoft PowerToys settings
- **vs2026/** - Visual Studio 2026 configuration and settings
- **windowsTerminal/** - Windows Terminal settings
- **winget/** - Windows Package Manager configuration
